Put PennyMon accessory PNGs in this folder.

Use transparent PNG files so the accessory can sit on top of PennyMon.

Recommended names:

- crown.png
- glasses.png
- ribbon.png
- hat.png
- flower.png

Example:

accessories/
  crown.png
  glasses.png
  ribbon.png

After adding new PNGs, tell Codex the filenames so they can be connected to the Accessories picker.

Put PennyMon color variants in this folder.

Each color folder should contain the same six mood filenames:

- angry.png
- calm.png
- excited.png
- happy.png
- sad.png
- worried.png

Example:

colors/
  pink/
    happy.png
    calm.png
    worried.png
    sad.png
    excited.png
    angry.png

  blue/
    happy.png
    calm.png
    worried.png
    sad.png
    excited.png
    angry.png
